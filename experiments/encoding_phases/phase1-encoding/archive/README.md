These versions represent the evolution of the encoding approach:
1. **V1**: Policy-first, strict rules
2. **V2**: Evidence-based from examples
3. **V3**: Blended policy + evidence
4. **CODEX**: Consolidated single-pass encoder

The current CODEX-PROMPT.md incorporates the best learnings from all versions.
